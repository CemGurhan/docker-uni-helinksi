Create a container from the image generated by this
Dockerfile with:

`docker run -it --name dockcon hello-docker sh`

And comment out this step if present:

`RUN touch additional.txt`

Move me into the `hello-docker` container with:

`docker cp ./additional.txt dockcon:/usr/src/app` 

As `/usr/src/app` is the wrkdir created for the
`hello-docker` image.
